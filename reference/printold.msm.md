# Print a fitted msm model object

Print a fitted msm model object (in old format, from msm 1.3.1 and
earlier)

## Usage

``` r
printold.msm(x, ...)
```

## Arguments

- x:

  Output from [`msm`](https://chjackson.github.io/msm/reference/msm.md),
  representing a fitted multi-state model object.

- ...:

  Other arguments to be passed to
  [`format`](https://rdrr.io/r/base/format.html).

## Details

See
[`print.msm`](https://chjackson.github.io/msm/reference/print.msm.md)
for a better and cleaner output format, and an explanation of the
change.
